## 核心项目架构
- api 定义接口
- build 用于构建的脚本
- CHANGELOG 变更内容
- cluster 集群构建
- cmd 命令行工具构建
- docs 文档
- hack 工具箱，各种脚本
- pkg 核心代码具体实现
- plugin 插件
- staging 暂存代码，会被逐步重构
- test 测试文件
- third_party 第三方依赖
- translations 国际化


## k8s.io
- api k8s中各种资源的数据结构

- apiextensions-apiserver 定义crd元资源
  - pkg/client/clientset 可以调用他注册crd实例

- apimachinery 对资源结构的定义，元数据

- client-go 客户端操作k8s资源
  - tools
    - record eventBroadcaster 封装事件广播器
  - util
    - workqueue 封装一些好用的队列
