## 配置文件
运行ansible时候会按顺序查找配置文件

1. ANSIBLE_CONFIG 环境变量指向的环境变量
2. ./ansible.cfg 执行命令目录下
3. ~/.ansible.cfg
4. /etc/ansible/ansible.cfg

常用配置项如下
- .inventory 需要连接管理的主机列表


# 变量
1. inventory文件定义主机变量
2. 通过/etc/ansible下文件定义主机变量
3. 通过ansible命令行传入变量
4. playbook文件内用vars
```
---
  - hosts:all
    vars:
      key: ansible
    tasks:
      - value {{key}}   
```
