## 包内规范
一般 __xxx__ 是特殊变量，可以被直接引用，但是有特殊用途

_x 或 __x 是private，外部不应该使用


## 入口
```
#! /usr/bin/python3
# 指定用哪个解释器运行
# -*- coding: utf-8 -*-

'一段注释'

__author__ = 'kanggege'

import subprocess

def test():
    res = subprocess.run(["ls","-al"])
    print(res)


# 即 main 函数入口
if __name__=='__main__':
    test()
```
